# SiftForward Design References

Use these files as visual references for the SiftForward website build.

## Files

- `siftforward-logo-reference.png` — Logo / wordmark direction.
- `siftforward-hero-reference.png` — Header and hero section reference.
- `siftforward-full-page-reference.png` — Full landing page structure reference.
- `siftforward-ai-in-action-reference.png` — Revised landing page reference with the AI in Action showcase.

## Suggested Project Location

Place this folder in the project at:

```txt
public/design-reference/
```

## Claude Code Instruction

Tell Claude Code:

```txt
Use the images in public/design-reference as visual references only. Do not treat them as final production assets unless instructed. Recreate the SiftForward logo and landing page style in clean Next.js/Tailwind code. Build a simple SVG/CSS logo inspired by the logo reference: scattered dots being sifted into a forward arrow, with the wordmark SiftForward. Keep the brand palette deep navy, charcoal, teal, cyan, and white.
```

## Important

The first build can use a recreated SVG/CSS logo instead of the PNG logo. Later, replace it with final exported SVG/PNG assets after the brand is finalized.
